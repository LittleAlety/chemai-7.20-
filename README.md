# ChemAI — 三草酸合铁(III)酸钾制备实验智能教学平台

[![Deploy](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://littlealety.github.io/chemai-7.20-/)


**ChemAI** 是一个面向大学化学实验教学的智能 Web 平台，以三草酸合铁(III)酸钾 K₃[Fe(C₂O₄)₃]·3H₂O 的制备实验为核心，集成 AI 问答、知识图谱、语料库检索、课前预习和实验报告评估等功能。

## 功能模块

| 模块 | 说明 |
|------|------|
| **AI 助手** | DeepSeek LLM RAG 智能体，基于 494 条 FAQ + 1335 条知识库 + 291 篇文献语料库检索回答 |
| **知识图谱** | 配位化学 / 氧化还原 / 分析化学 / 物理化学四大知识方向的关联网络 |
| **语料库** | 291 篇中英文文献知识清单，支持 PDF 上传和在线检索 |
| **课前预习** | 多轮对话预习 + 自适应习题检测 |
| **视频资源** | 实验操作视频合集 + B站链接 |
| **报告评估** | 实验报告检测评估系统 |
| **科普探索** | 蓝晒工艺 / 光化学 / 配合物拓展知识 |

## 技术架构

```
用户提问
  ↓
① 检索知识清单（FAQ 494条 + KB 1335条 + 语料库 291篇）
  ↓
② 具体查找文献（FAQ 关键词 + BM25 全文 + Corpus 三层检索）
  ↓
③ 有结果？──YES──→ DeepSeek LLM 基于文献精准回答
  ↓                  
  NO                 
  ↓                  
④ 接入网络浏览（Bing / CNKI / Google Scholar / B站 等 7 个搜索引擎）
```

- **前端**: 纯 HTML/CSS/JS，单文件 SPA，暗色主题
- **AI 引擎**: DeepSeek API（兼容 OpenAI）RAG 模式
- **检索引擎**: BM25 全文搜索 + FAQ 关键词匹配 + 语料库打分
- **部署**: GitHub Pages 静态托管，零后端依赖
- **知识库**: 494 FAQ + 1335 KB + 291 文献语料

## 快速开始

### 本地运行

```bash
# 启动 HTTP 服务器
cd chemai-7.20-
python -m http.server 8080

# 浏览器打开
http://localhost:8080/assistant.html
```

### 启用 AI 智能体

1. 打开 `assistant.html`，在左侧边栏 **🤖 LLM 配置** 面板中：
   - 粘贴你的 DeepSeek / OpenAI API Key
   - 选择模型（默认 DeepSeek V3）
   - 点击 💾 保存
2. 状态显示 `✅ LLM 就绪` 即可开始提问
3. 如不配置 Key，自动回退纯本地检索模式

> ⚠️ 代码中不含硬编码 API Key，每个用户需自行配置。部署到公开网站时不会有 Key 泄露风险。

### 支持的 LLM 模型

| 模型 | 供应商 | 说明 |
|------|--------|------|
| `deepseek-chat` | DeepSeek | 推荐，性价比高 |
| `deepseek-reasoner` | DeepSeek | 推理增强 |
| `gpt-4o` | OpenAI | 最强性能 |
| `gpt-4o-mini` | OpenAI | 快速轻量 |
| `claude-3-5-sonnet` | Anthropic | 平衡性能 |

## 实验知识覆盖

三草酸合铁(III)酸钾 K₃[Fe(C₂O₄)₃]·3H₂O 制备实验的 **12 章完整知识体系**：

| 章节 | 内容 | 知识点 |
|:----:|------|--------|
| Ch1 | 实验概述与背景 | 配位化学史、维尔纳理论、教学目标 |
| Ch2 | 化合物性质详解 | 物理/化学性质、颜色成因、稳定性 |
| Ch3 | 制备原理深度解析 | 三步反应方程式、试剂选择原理 |
| Ch4 | 操作步骤完全指南 | 六步主线、温度控制、抽滤结晶 |
| Ch5 | 配合物性质实验 | KMnO₄滴定、组成测定、离子检验 |
| Ch6 | 光化学性质 | LMCT机理、蓝晒工艺、光化学剂量计 |
| Ch7 | 晶体场理论 | d-d跃迁、CFSE、光谱化学序列 |
| Ch8 | 安全规范与废液处理 | MSDS、个人防护、废液分类 |
| Ch9 | 教学反思与改进 | 实验设计优化、产率提升策略 |
